= Installing OData V2 Tooling
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

To install APIkit for OData v2 plugin in Anypoint Studio:

. Select *Help* > *Install New Software* and click *Add*.
+
image::add-update-site.png["Window for installing new software with the *Add* button highlighted"]
. In *Name*, type `APIkit for ODATA v2 Update Site`.
. In *Location*, type the URL of the site: `+https://studio.mulesoft.org/s4/apikit-for-odata/+`.
. Click *Add*.
+
image::add-repository-odata.png["Window for adding a repository with the *Add* button, and the name and the location fields highlighted"]
. Select *Anypoint ODATA Plugin*, and click *Next*.
+
image::select-odata-extension.png["Window highlighting the Anypoint OData plugin field and the Next button"]
. Accept the terms and conditions of the product, and click *Finish*.
. Restart Studio to complete the installation.
