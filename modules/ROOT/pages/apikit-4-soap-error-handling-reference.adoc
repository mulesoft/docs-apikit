APIkit offers an error handling mechanism for your application process. This article lists the typed errors for APIkit for SOAP. 

== APIkit for SOAP Error Types

APIkit for SOAP defines two error types:

[%header]
|===
|Error Type |Description

|APIKIT-SOAP:ROUTER
|Routing exception

|APIKIT-SOAP:FAULT
|Fault exception
|===

== APIkit for SOAP Error Handling

APIkit for SOAP scaffolder does not generate error-handling code.
