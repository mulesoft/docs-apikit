This section showcases the implementation example of a GraphQL service to query data using APIkit for GraphQL.

To run the set of examples described below, download and run the *apikit-graphql-example* application, which loads a sample dataset in memory for testing purposes. Get more information about https://graphql.org/learn/serving-over-http/[the format of the queries sent through an HTTP POST request^].
